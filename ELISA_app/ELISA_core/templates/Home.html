Introduction

Welcome to the ELISA App!

This app was developed to generate graphs and results from ELISA reader data in a convenient and streamlined manner.

Check out "Getting Started" to start processing your data.

Getting Started

The menu on the left contains all the steps for data processing in order.

Steps "Input Data", "Plate Layout", and "Dillutions" are for data input. Steps starting from "Visualize Data" will not be available unless all data has been input.

Click on "Input Data" to get started!

Please refer to the User Manual for detailed instructions on how to use the app.
